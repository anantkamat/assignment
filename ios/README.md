# Steps to install

## Clone the repo
```bash
git clone https://github.com/anantkamat/room-booking.git
```
## Install dependencies
```bash
npm install
```

## For IOS
```bash
npm install
```
## Run bundle in separate tab
```bash
react-native start
```

## Build app
```bash
react-native run-android
```
